# Abu Awaish Portfolio

A personal portfolio website for **Abu Awaish**, a Salesforce Developer at Tata Consultancy Services with a strong interest in Data Science, Machine Learning, and full-stack development.

## About

- Clean single-page portfolio built with HTML, Tailwind CSS, and JavaScript.
- Showcases experience, skills, projects, education, interests, and contact details.
- Includes animated hero section, custom cursor, particle background, and smooth scrolling.

## Featured Projects

- **Salesforce DB Admin Tool** — Streamlit app for Salesforce data management and SOQL support.
- **PDF Insight Analyzer** — AI/NLP web app for turning PDFs into searchable knowledge bases.
- **Myntra Product Scraper** — Flask and Selenium-based ecommerce scraping dashboard.
- **PyAwaish** — Python package for MySQL-powered web apps with CRUD and REST features.
- **MongoDB Manager** — Streamlit interface for MongoDB collection management.
- **Flight Fare Predictor** — Machine learning model for predicting airfare with high accuracy.
